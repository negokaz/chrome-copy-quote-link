### ‚Ä£ Copy link via Shortcut Key

Default shortcut key is follows. 

- **Windows**: `Ctrl`+`Shift`+`X`
- **Mac**: `Command`+`Shift`+`X`
- **Linux**: `Ctrl`+`Shift`+`X`

You can change the setting via `chrome://extensions/shortcuts`

![](docs/images/usage-shortcut-key.png)
